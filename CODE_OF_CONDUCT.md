# Code of conduct

Aria Focus contributors must communicate respectfully, welcome different
experience levels, give actionable technical feedback, and avoid harassment,
discrimination, threats, sexualized conduct, or publication of private
information.

Maintainers may edit or remove contributions and restrict participation when
behaviour harms the community. Report conduct concerns privately through the
repository owner's GitHub profile. Reports will be reviewed confidentially and
fairly, with consequences proportionate to context and severity.

This policy applies in project spaces and when someone is publicly representing
the project.
